**Superstore Sales Analysis**

This repository contains an Excel-based sales analytics project (Superstore Sales Data V2 File) using the Superstore dataset. The focus is on creating dynamic insights and visualizations with PivotTables, formulas, and slicers.

**Features**

Average Orders per Day:
Calculated via Excel formulas to provide an at-a-glance metric of daily order activity.
(Note: Currently static; future enhancement may include dynamic updates with slicers.)

**PivotTables & Slicers:**
Explore sales by region, product, and time period. All charts and tables update dynamically when slicers are used.

**Clean & Maintainable:**
Raw data is separate from analysis tables, ensuring reliability and ease of updates.

**Usage**

Open the Excel file and review the raw data table.

Use PivotTables and slicers to filter and analyze sales data.

The Average Orders per Day metric provides a quick snapshot of overall daily order trends.

**Dataset**

Dataset used: Superstore Sales

**Future Enhancements**

Make Average Orders per Day update dynamically with slicers and filters.
